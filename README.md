# Suraj Kumar — Developer Portfolio

This is a Vite + React developer portfolio scaffold built with Tailwind CSS and Framer Motion. It includes:

- Dark theme with glassmorphism cards
- Smooth scrolling and sticky navbar
- Framer Motion animations and subtle particles background
- Reusable components and responsive layout

Quick start:

```bash
# Install deps
npm install

# Start dev server
npm run dev
```

Replace `/resume.pdf` with your resume. Update project links and content in `src/components/Projects.jsx`.
